## You can test it locally

### Prerequisites

- Install [Docker](https://www.docker.com/)
- Install [Docker Compose](https://docs.docker.com/compose/install/)

### Usage

Run the web service with docker

```
docker-compose up 
```

Check http://localhost:8080/admin/backfill
